# stiffness-method-plane-truss-jupyter
Computational implementation of the stiffness method for the matrix analysis of a plane truss in Jupyter Notebook.

This repository contains the official version 1.0.0 of the Jupyter Notebook used for the computational implementation of the stiffness method for a plane truss. Contributions may be proposed through pull requests. Accepted changes will be incorporated into subsequent versions.

Este repositorio contiene la versión oficial 1.0.0 del notebook de Jupyter utilizado para la implementación computacional del método de rigidez para una cercha plana. Las contribuciones pueden proponerse mediante pull requests. Los cambios aceptados serán incorporados en versiones posteriores.
